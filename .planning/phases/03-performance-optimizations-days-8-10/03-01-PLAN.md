<objective>
Create `AzureSDKManager` — a singleton that creates `DefaultAzureCredential` once at startup,
caches Azure SDK clients (Compute, Network, Storage, Cosmos, etc.), and configures connection
pooling for both sync and async clients. Wire the manager into FastAPI's lifespan context so
credentials are initialized on startup and cleaned up on shutdown.

Purpose: 6+ agents currently call `DefaultAzureCredential()` independently on every instantiation,
incurring repeated token acquisitions. A singleton eliminates this overhead (NFR-PRF-03: <100ms
after first request) and enables connection reuse (NFR-PRF-04: ≥80% pool efficiency).

Output: `utils/azure_client_manager.py`, updated `main.py` lifespan, unit tests.
</objective>

**Class design:**

```python
class AzureSDKManager:
    _instance: ClassVar[Optional["AzureSDKManager"]] = None
    _lock: ClassVar[asyncio.Lock]  # create in __init_subclass__ or lazily

    def __init__(self):
        self._credential: Optional[DefaultAzureCredential] = None
        self._async_credential: Optional[AsyncDefaultAzureCredential] = None
        self._sync_clients: Dict[str, Any] = {}   # keyed by "type:subscription_id"
        self._async_clients: Dict[str, Any] = {}
        self._initialized: bool = False
        self._close_lock = asyncio.Lock()
```

**Singleton pattern** (thread-safe for async context):
```python
    @classmethod
    def get_instance(cls) -> "AzureSDKManager":
        if cls._instance is None:
            cls._instance = cls()
        return cls._instance
```

**Credential creation** (lazy, cached, with persistent token cache):
```python
    def get_credential(self) -> DefaultAzureCredential:
        if self._credential is None:
            # Exclude slow credential sources — App Service will use ManagedIdentityCredential
            # EXCLUDE_CREDENTIALS env var can add more exclusions
            excluded = os.getenv("AZURE_CREDENTIAL_EXCLUDE", "").split(",")
            # Attempt persistent token cache (TECH-AZ-05); falls back gracefully if the
            # msal-extensions persistence extension is not installed in this environment.
            try:
                from azure.identity import TokenCachePersistenceOptions
                cache_opts = TokenCachePersistenceOptions(name="gcc-demo")
            except Exception:
                cache_opts = None  # persistence extension unavailable — use in-memory cache
            self._credential = DefaultAzureCredential(
                cache_persistence_options=cache_opts,
                exclude_workload_identity_credential=False,
                exclude_developer_cli_credential="DeveloperCli" in excluded,
            )
        return self._credential

    def get_async_credential(self) -> AsyncDefaultAzureCredential:
        if self._async_credential is None:
            self._async_credential = AsyncDefaultAzureCredential()
        return self._async_credential
```

**Sync client factory** with connection pooling (transport configured via `azure-core` `HttpTransport`):
```python
    def get_compute_client(self, subscription_id: str) -> ComputeManagementClient:
        key = f"compute:{subscription_id}"
        if key not in self._sync_clients:
            transport = RequestsTransport(
                connection_pool_size=10,
                max_connections=20,
            )
            self._sync_clients[key] = ComputeManagementClient(
                credential=self.get_credential(),
                subscription_id=subscription_id,
                transport=transport,
            )
        return self._sync_clients[key]
```

**Lifecycle methods:**
```python
    async def initialize(self) -> None:
        """Warm up credentials (call on startup to pre-acquire token)."""
        self._initialized = True
        # Optionally pre-warm credential by requesting a test token
        # Don't fail startup if this fails

    async def aclose(self) -> None:
        """Close all async clients gracefully."""
        async with self._close_lock:
            for client in self._async_clients.values():
                try:
                    await client.close()
                except Exception:
                    pass
            self._async_clients.clear()
            if self._async_credential:
                try:
                    await self._async_credential.close()
                except Exception:
                    pass
            self._initialized = False

    def is_initialized(self) -> bool:
        return self._initialized
```
