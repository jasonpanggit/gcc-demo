<objective>
Update all 6 agents to source Azure credentials from `AzureSDKManager` instead
of constructing `DefaultAzureCredential()` or `AsyncDefaultAzureCredential()` independently.
This eliminates redundant token acquisitions, enabling credential reuse and connection pool
sharing (PRF-03, NFR-PRF-03).

Plan 03-01 identified 6 agents with independent credential creation:
- 4 sync agents: `azure_ai_agent`, `os_inventory_agent`, `software_inventory_agent`, `openai_agent`
- 2 async agents: `monitor_agent` (line 479: AsyncDefaultAzureCredential), `domain_sub_agent` (line 254: AsyncDefaultAzureCredential)

All 6 are migrated here. The async agents use `get_azure_sdk_manager().get_async_credential()`
instead of `get_azure_sdk_manager().get_credential()` to preserve async context correctness.

Purpose: Credential warm-up now happens once at startup. Agents that previously each triggered
their own MSAL auth flow now share the pre-warmed token from the singleton.

Output: Updated agents with shared credential, 2 integration-style tests verifying credential sharing.
</objective>

```python
class AzureSDKManager:
    @classmethod
    def get_instance(cls) -> "AzureSDKManager": ...
    def get_credential(self) -> DefaultAzureCredential: ...
    def get_async_credential(self) -> AsyncDefaultAzureCredential: ...

    def get_monitor_client(self, subscription_id: str) -> MonitorManagementClient: ...
    async def initialize(self) -> None: ...
    async def aclose(self) -> None: ...
    def is_initialized(self) -> bool: ...

def get_azure_sdk_manager() -> AzureSDKManager:
    return AzureSDKManager.get_instance()
```

```python
@pytest.mark.unit
def test_agents_share_same_credential():
    """Two agents using get_azure_sdk_manager() must receive the same credential object."""
    from unittest.mock import patch, MagicMock

    # Reset singleton for clean test
    AzureSDKManager._instance = None

    mock_cred = MagicMock()
    with patch("utils.azure_client_manager.DefaultAzureCredential", return_value=mock_cred):
        manager = get_azure_sdk_manager()
        cred1 = manager.get_credential()
        cred2 = manager.get_credential()

    assert cred1 is cred2, "Credential must be the same object — no re-instantiation"
```
